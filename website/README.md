# AISplinter website

Static landing page with animated **crystal planet** hero (diamond bursts → splinter shards → gravity).

## Preview locally

```bash
cd aisplinter-code/website
npx --yes serve .
# open http://localhost:3000
```

Or open `index.html` directly in a browser (some features work; `config.js` must load).

## Edit links

`config.js` — GitHub, docs, donate URL, other projects.

## Deploy

Host on **GitHub Pages** ( `/website` folder or `gh-pages` branch ), **Cloudflare Pages**, or **Netlify` — static files only.

Point `aisplinter.dev` at this folder when you have a domain.
